## Présentation

Un dashboard météo qui consomme l'API OpenWeatherMap et affiche les données sous forme de graphiques interactifs.

## Fonctionnalités

- Prévisions sur 7 jours
- Graphiques de températures et précipitations
- Recherche par ville
- Géolocalisation automatique

## Technologies

- React, Chart.js
- OpenWeatherMap API
- Tailwind CSS
