## Présentation

Une application web de gestion de tâches pensée pour la productivité quotidienne.

## Fonctionnalités

- Création et organisation de tâches par catégorie
- Authentification utilisateur
- Rappels par email
- Interface responsive

## Technologies

- Next.js, TypeScript, Tailwind CSS
- PostgreSQL, Prisma
- NextAuth.js
