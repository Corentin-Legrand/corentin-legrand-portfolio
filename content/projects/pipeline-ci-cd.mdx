## Présentation

Un pipeline d'intégration et de déploiement continu pour automatiser les livraisons d'une application web.

## Fonctionnalités

- Tests automatisés à chaque push
- Build et push d'image Docker
- Déploiement automatique sur VPS
- Notifications Slack en cas d'échec

## Technologies

- GitHub Actions
- Docker, Docker Compose
- Linux, Nginx
