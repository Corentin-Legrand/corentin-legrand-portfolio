## Présentation

Algochal est une plateforme web d'entraînement à l'algorithmique orientée Pascal, pensée sur le modèle de plateformes comme LeetCode.
Le projet permet de créer des exercices, de les consulter et de les résoudre directement depuis une interface dédiée.
Une attention particulière a été portée à la sécurité, avec la mise en place d'un système de sandbox côté serveur pour exécuter le code utilisateur de manière isolée.

## Fonctionnalités

- Création et gestion d'exercices Pascal
- Interface de résolution directement depuis l'application
- Exécution sécurisée du code sur le serveur
- Isolation des environnements d'exécution via sandbox
- Architecture séparée entre front-end Angular et API REST ASP.NET
